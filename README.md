- **Theme**: Crimson red and black with dark-mode focus
- **Aesthetic**: Command-line inspired, minimal, and powerful
- **Typography**: JetBrains Mono for headings, Inter for body text
- **Animations**: Smooth fade-ins, glowing effects, and terminal-style backgrounds
- **Layout**: Structured card-based design similar to Voxlis.net

## Technology Stack

- **HTML5** - Semantic markup

## Usage

1. Open `index.html` in a modern web browser
2. Navigate through tool categories using the navigation menu
3. Click on tool cards to launch interactive modal interfaces
4. Use tools for educational purposes and network diagnostics
